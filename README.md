Quiz Game in C++

Description

This project is a simple console-based Quiz Game developed using C++.
It asks multiple-choice questions to the user, takes input, and calculates the final score based on correct answers.

Features

- Multiple-choice questions
- User input handling
- Score calculation
- Instant result display
- Simple and interactive console interface

Technologies Used

- Language: C++
- Compiler: GCC / MinGW
- IDE: VS Code

 How to Run

1. Open the project in VS Code
2. Compile the program:
   g++ quiz.cpp -o quiz
3. Run the program:
   ./quiz

📂 Project Structure

- "quiz.cpp" → Main source code
- "README.md" → Project documentation

 Output

- Displays questions one by one
- User selects an option (A/B/C/D)
- Final score is shown at the end

Objective

The main objective of this project is to understand:

- Basic C++ programming
- Conditional statements
- Loops
- User input/output handling

⚠️ Limitations

- Fixed set of questions
- No graphical interface (console-based only)

🔮 Future Improvements

- Add more questions dynamically
- Store questions in files
- Add timer for each question
- Create GUI version
